# Super Escola — Landing Page

## Visão Geral
Landing page de marketing para o sistema de gestão académica "Super Escola", construída em PHP puro.

## Estrutura do Projeto
- `public/index.php` — página principal (inclui todos os componentes)
- `public/components/` — secções da página (header, hero, features, etc.)
- `public/assets/css/style.css` — estilos completos
- `public/assets/js/main.js` — interatividade (menu mobile, FAQ, scroll)
- `server.php` — servidor de desenvolvimento PHP

## Como Correr
O workflow "Start application" corre: `php -S 0.0.0.0:5000 -t public`

## Tecnologias
- PHP 8.2
- CSS puro (sem frameworks)
- JavaScript vanilla
